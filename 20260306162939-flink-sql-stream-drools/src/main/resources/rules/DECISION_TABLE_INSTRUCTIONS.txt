Drools Decision Table Instructions
=================================

This file provides instructions for creating a Drools Decision Table (.xls format)
for the Flink Transaction Tagging application.

Creating a Decision Table
--------------------------

1. Open Microsoft Excel or compatible spreadsheet software
2. Create a new workbook and save as "decision-table.xls" (must be .xls, not .xlsx)
3. Follow the structure below to define your rules

Important Notes
---------------

1. File Format: Must be .xls (Excel 97-2003), NOT .xlsx
2. Empty Cells: Empty cells mean that condition/action is not part of that rule
3. Row Order: Follow the exact order shown above
4. Syntax: Use exact Drools syntax in condition and action cells
5. Multiple Conditions: Add more CONDITION columns for complex rules
6. Multiple Actions: Add more ACTION columns for multiple actions per rule

Testing Your Decision Table
----------------------------

After creating your decision-table.xls file:

1. Save it to: src/main/resources/rules/decision-table.xls
2. Run the application with decision-table mode:
   
   java -jar target/flink-transaction-tagging-1.0.0.jar sql input.csv output.csv decision-table

3. Check logs for compilation errors
4. Verify rules are firing correctly in the output

Troubleshooting
---------------

If you encounter errors:

1. Verify the file is saved as .xls (not .xlsx)
2. Check that all import statements are correct
3. Ensure condition and action syntax is valid Drools
4. Look for empty cells that should have values
5. Review the Drools documentation for detailed syntax rules

Advanced Features
-----------------

You can add more complex rules by:
- Adding multiple CONDITION columns
- Adding multiple ACTION columns
- Using complex Drools expressions
- Setting rule priorities (salience)
- Adding metadata to rules

For more information, refer to:
- Drools Documentation: https://docs.drools.org/latest/drools-docs/html_single/index.html
- Decision Table Guide: https://docs.drools.org/latest/drools-docs/html_single/index.html#drg-tables-decision-tables
